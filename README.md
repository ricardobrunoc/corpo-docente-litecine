# Frame de divulgação dos professores

Arquivos principais:
- index.html
- style.css

Como usar no site:
```html
<iframe src="professores_frame/index.html" style="width:100%; height:760px; border:0;" title="Docentes do curso"></iframe>
```

Fotos:
- Coloque as imagens em assets/professores/
- Use os nomes de arquivo já referenciados no HTML.
- Se uma foto não existir, aparece o fallback com iniciais.
